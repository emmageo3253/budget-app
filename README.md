## Note
This project is under active development and is part of a larger personal portfolio.

A personal finance dashboard built with **Next.js** and **Supabase** to track weekly budgets, savings goals, and spending across categories.

This project was built to explore real-world budgeting logic, theme-aware UI design, and persistent user state.

## Features
- Weekly budget tracking
- Savings, student loans, emergency, and extra money buckets
- Editable goals with donut-style progress charts
- Theme-aware UI (Soft, Mint, Cool, Dark)
- Supabase authentication and database persistence
- Responsive, component-based design

## Tech Stack
- **Next.js** (App Router)
- **React**
- **Supabase** (Auth + Database)
- **CSS Variables** for theming
- TypeScript
